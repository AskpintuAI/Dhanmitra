# Dhanmitra — GitHub Pages Deploy Package

## Quick Steps
1. Naya GitHub repo banao naam: **dhanmitra** (exact)
2. Is zip ko EXTRACT karke saari files (folders sahit) repo me upload karo
3. Settings → Pages → Source: Deploy from branch → main → / (root) → Save
4. Live URL: https://YOUR_USERNAME.github.io/dhanmitra/

## Firebase
Firebase Console → Authentication → Authorized domains → add: YOUR_USERNAME.github.io
